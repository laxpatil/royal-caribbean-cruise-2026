# 🚢 Cruise Day Planner

Interactive day planner for a Caribbean cruise (April 12–19) with a toddler.

## Ports Covered
- 🇲🇽 **Costa Maya, Mexico** — Beach day with pier splash area & Mahahual
- 🇭🇳 **Roatán, Honduras** — West Bay Beach, snorkeling & Gumbalimba Park

## Features
- Toddler-friendly itineraries with timeline view
- Packing checklists for each port
- Tips & practical advice for traveling with a 16-month-old
- Interactive tab navigation between ports

## Usage
Just open `index.html` in any browser — it's a single self-contained HTML file, no dependencies needed.

Built with ❤️ for a family cruise adventure!
