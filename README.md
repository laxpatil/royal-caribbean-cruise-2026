- 👶 **Toddler-optimized** — calm beaches, nap windows, shallow water spots
- 🎒 **Packing checklists** — port-specific gear lists

## 🚀 Usage

Simply open the [**live site**](https://pages.github.pie.apple.com/lpatil/cruise-planner/) on your phone or laptop — no installation needed!

Or clone and open locally:

```bash
git clone https://github.pie.apple.com/lpatil/cruise-planner.git
open cruise-planner/index.html
```

## 👶 Toddler Travel Tips

> - Apply **reef-safe SPF 50** every 90 minutes
> - Pack **swim diapers** — required at all beaches & pools
> - Schedule activities around **midday nap time** (11:30 AM – 1 PM)
> - **USD is accepted** at both Costa Maya and Roatán
> - Always eat **breakfast on the ship** before disembarking
